# 🚀 自媒体排版大师后续迭代计划 (ROADMAP)

为了将本项目打造成真正能解决自媒体创作者痛点的生产级开源工具，我们规划了以下迭代路线图。欢迎开发者共同建设！

---

## 📅 第一阶段：高价值体验优化 (短期计划 - 🌟 极力推荐优先实现)

- [ ] **📊 实时平台规格校验与字数统计**
  - **功能**：在预览区上方增加实时状态条。
  - **规格约束**：
    - **小红书**：标题字数限制指示（0/20字，超限预警变红）、正文字数进度条（0/1000字）。
    - **短视频脚本**：口播字数统计，自动换算预估口播时长（按约 200 字/分钟估算，例如：`预估时长 1:23`）。
    - **微信公众号**：总字数统计及预计阅读时间（例如：`2,341 字 · 预计阅读 8 分钟`）。
  - **价值**：避免用户生成内容后因超限在平台发布失败，提供即时反馈。

- [ ] **📋 小红书一键纯文本复制通道**
  - **功能**：当平台选择“小红书”时，一键复制按钮自动切换为纯文本模式。
  - **规则**：自动去除所有 HTML 标签、内联样式以及无意义占位符，仅保留纯文本、Emoji 表情、换行和 `#话题标签`。
  - **价值**：小红书后台编辑器不支持 HTML 粘贴，纯文本复制可解决小红书粘贴带 HTML 代码的痛点。

- [ ] **🌙 全局暗黑模式支持 (Dark Mode)**
  - **功能**：基于 Tailwind 或 CSS Variables，实现极简丝滑的暗黑/亮色主题切换。
  - **价值**：提升开发者和自媒体人的夜间使用体验，增强开源产品的极客感和品质感。

---

## 📅 第二阶段：核心差异化功能 (中期计划 - 提升产品独特性)

- [ ] **📱 一稿多发：一键多平台并行生成预览**
  - **功能**：用户只需输入一份草稿，AI 同时生成**微信公众号**、**小红书图文**、**短视频口播脚本**三个版本的排版效果。
  - **布局**：前端提供多栏左右并排预览。
  - **价值**：真正解决多平台分发场景下的“一稿多改”低效难题，形成降维打击。

- [ ] **💾 纯前端历史记录与版本管理**
  - **功能**：使用 `localStorage` 或 `IndexedDB` 存储用户最近的生成历史（如最近 20 篇）。
  - **特性**：支持一键回滚历史版本、双版本文本差异对比 (Diff)。
  - **价值**：免去繁琐的账号注册与后端数据库依赖，保障用户数据隐私。

- [ ] **📤 结构化文件导出引擎**
  - **功能**：
    - 微信公众号：一键导出为带完整 CSS 内联样式的 `.html` 富文本文件。
    - 小红书：一键导出为 `.txt` 文档。
    - 短视频脚本：一键导出为包含分镜与口播对比的标准 `.docx` / `.xlsx` 表格。

- [ ] **🔑 零门槛部署：API Key 用户自定义模式**
  - **功能**：设置中心增加用户自定义 API Key 输入框（支持 DeepSeek、OpenAI、月之暗面等常用 API 地址）。
  - **存储**：安全加密存储于用户的本地浏览器 `localStorage`。
  - **价值**：让开源用户能够一键克隆项目并立即在 Vercel 免费托管，无需项目发起者承担 API 额度。

## 📅 第三阶段：生态建设与 AI 增强 (长期计划 - 构建完整闭环)

- [ ] **🖼️ AI 智能配图与封面图生成**
  - **功能**：结合 Midjourney、DALL-E 或 SD，根据文章主旨自动生成匹配小红书的 3:4 比例或微信的 16:9 比例封面图与插图。

- [ ] **🌐 i18n 国际化支持**
  - **功能**：支持中英双语界面，拓展海外 Creator 创作者用户群体。

- [ ] **📈 自媒体文案 SEO 与网感健康评分**
  - **功能**：自动对生成文案进行“标题党程度”、“情绪共鸣度”、“敏感词检测”以及“阅读通顺度”评分，并提供针对性的 AI 修改建议。
